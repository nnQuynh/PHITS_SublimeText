## Manual Installation
1.  Open Sublime Text.
2.  Go to **Preferences** > **Browse Packages...** [13].
3.  Inside the Packages folder, create a new folder named `YourPackageName`.
4.  Copy the package files into this new folder.
5.  Restart Sublime Text.
